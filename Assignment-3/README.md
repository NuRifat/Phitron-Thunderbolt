# 🏦 Banking System (CLI + OOP + SQLite)

A role-based banking system built using Python OOP and SQLite.  
It supports Admin, Manager, Employee, and Customer roles with different permissions.

---

## 🚀 Features

### 👤 User System
- User registration with roles (Admin / Manager / Employee / Customer)
- Login system
- Role-based access control (RBAC)

### 💰 Banking Features (Customer)
- Deposit money
- Withdraw money
- Transfer money using email
- Check balance
- View transaction history

### 👑 Admin Features
- View all users
- Delete users
- View all accounts

### 🧑‍💼 Manager Features
- Create employees
- View employees

### 👨‍💻 Employee Features
- View customers
- Deposit money for customers

---

## 🛠️ Tech Stack
- Python 3
- SQLite3
- OOP (Object-Oriented Programming)
- CLI Interface
